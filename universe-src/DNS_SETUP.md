# Настройка DNS для домена civilxuniverse.ru

## Проблема
Домен `civilxuniverse.ru` не резолвится в DNS, поэтому браузер не может найти сервер.

## Решение: Настройка DNS записей

### Шаг 1: Определите, где зарегистрирован домен

Домен должен быть зарегистрирован у регистратора (например, REG.RU, Timeweb, Beget и т.д.).

### Шаг 2: Настройте DNS записи

Войдите в панель управления вашего регистратора домена и настройте следующие DNS записи:

#### Вариант A: Использование A-записи (рекомендуется)

```
Тип: A
Имя: @ (или оставить пустым, или указать civilxuniverse.ru)
Значение: 95.163.230.61
TTL: 3600 (или автоматически)
```

```
Тип: A
Имя: www
Значение: 95.163.230.61
TTL: 3600 (или автоматически)
```

#### Вариант B: Использование CNAME для www

```
Тип: A
Имя: @ (или оставить пустым)
Значение: 95.163.230.61
TTL: 3600
```

```
Тип: CNAME
Имя: www
Значение: civilxuniverse.ru
TTL: 3600
```

### Шаг 3: Проверка DNS записей

После настройки DNS записей подождите 5-30 минут (время распространения DNS) и проверьте:

```bash
# Windows
nslookup civilxuniverse.ru

# Linux/Mac
dig civilxuniverse.ru +short
# или
host civilxuniverse.ru
```

Должен вернуться IP адрес: `95.163.230.61`

### Шаг 4: Проверка работы сайта

После того, как DNS записи настроены и распространились:

1. Откройте браузер
2. Перейдите на `http://civilxuniverse.ru`
3. Должна открыться страница CivilX.Universe

## Временное решение для тестирования

Если нужно протестировать сайт до настройки DNS, можно использовать файл `hosts`:

### Windows:
1. Откройте файл `C:\Windows\System32\drivers\etc\hosts` от имени администратора
2. Добавьте строку:
   ```
   95.163.230.61    civilxuniverse.ru
   95.163.230.61    www.civilxuniverse.ru
   ```
3. Сохраните файл
4. Откройте `http://civilxuniverse.ru` в браузере

### Linux/Mac:
1. Откройте файл `/etc/hosts` с правами root:
   ```bash
   sudo nano /etc/hosts
   ```
2. Добавьте строку:
   ```
   95.163.230.61    civilxuniverse.ru
   95.163.230.61    www.civilxuniverse.ru
   ```
3. Сохраните файл (Ctrl+O, Enter, Ctrl+X)
4. Откройте `http://civilxuniverse.ru` в браузере

**Примечание:** Это решение работает только на вашем компьютере. Для доступа других пользователей нужно настроить DNS записи.

## Где настроить DNS записи (примеры регистраторов)

### REG.RU
1. Войдите в личный кабинет
2. Перейдите в раздел "Домены" → "Мои домены"
3. Выберите домен `civilxuniverse.ru`
4. Перейдите в "DNS-серверы и зона"
5. Добавьте A-записи как указано выше

### Timeweb
1. Войдите в панель управления
2. Перейдите в "Домены" → "Управление DNS"
3. Выберите домен `civilxuniverse.ru`
4. Добавьте A-записи

### Beget
1. Войдите в панель управления
2. Перейдите в "Домены" → "Управление DNS"
3. Выберите домен
4. Добавьте записи

### Другие регистраторы
Процесс аналогичен - найдите раздел управления DNS для вашего домена и добавьте A-записи.

## Проверка после настройки

После настройки DNS и ожидания распространения (5-30 минут):

```bash
# Проверка DNS
nslookup civilxuniverse.ru
# Должен вернуться: 95.163.230.61

# Проверка сайта
curl http://civilxuniverse.ru/
# Должен вернуться HTML код страницы
```

## Если DNS не работает после настройки

1. Проверьте, что DNS записи сохранены правильно
2. Подождите больше времени (до 24 часов в редких случаях)
3. Очистите DNS кэш:
   - Windows: `ipconfig /flushdns`
   - Linux: `sudo systemd-resolve --flush-caches` или `sudo service network-manager restart`
   - Mac: `sudo dscacheutil -flushcache`
4. Проверьте, что домен действительно зарегистрирован и активен

## Готово!

После настройки DNS записей домен `http://civilxuniverse.ru` будет работать для всех пользователей.
