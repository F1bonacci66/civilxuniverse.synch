# Статус DNS для civilxuniverse.ru

## Текущие данные домена:
- **Доменное имя**: civilxuniverse.ru
- **Владелец**: www-root
- **IP-адрес**: 95.163.230.61

## Что нужно проверить:

### 1. В панели управления доменом проверьте:

Убедитесь, что созданы DNS записи типа A:

**Запись 1:**
- Тип: `A`
- Имя: `@` (или пустое, или `civilxuniverse.ru`)
- Значение: `95.163.230.61`
- TTL: `3600`

**Запись 2:**
- Тип: `A`
- Имя: `www`
- Значение: `95.163.230.61`
- TTL: `3600`

### 2. Если записи не созданы:

В панели управления доменом:
1. Найдите раздел "DNS записи" или "Управление DNS"
2. Добавьте две A-записи как указано выше
3. Сохраните изменения

### 3. Проверка после настройки:

```bash
nslookup civilxuniverse.ru
# Должен вернуться: 95.163.230.61
```

### 4. Время распространения:

DNS записи распространяются от 5 минут до 24 часов (обычно 5-30 минут).

## Если DNS уже настроен:

Если в панели управления DNS записи уже созданы, просто подождите 5-30 минут для их распространения.

## Проверка работы:

После распространения DNS откройте в браузере:
- `http://civilxuniverse.ru`
- `http://www.civilxuniverse.ru`

Оба должны открыть страницу CivilX.Universe.
