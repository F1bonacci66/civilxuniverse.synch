## Выполните на сервере:

```bash

# (с локальной машины)
scp fix-cors-debug.sh root@95.163.230.61:/tmp/

# На сервере
chmod +x /tmp/fix-cors-debug.sh
bash /tmp/fix-cors-debug.sh
```

Скрипт:
1. Проверит текущую конфигурацию
2. Проверит, что происходит с OPTIONS запросами
3. Применит исправленную конфигурацию
4. Проверит результат
