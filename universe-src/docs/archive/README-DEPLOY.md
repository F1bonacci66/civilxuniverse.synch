# Инструкция по развёртыванию на civilx.ru

## Важно!

После сборки статические файлы будут в папке `out/`. Их нужно скопировать в папку `/universe/` на сервере.

## Шаги развёртывания:

1. **Соберите приложение:**
```bash
cd civilx-website/universe/universe
npm install
npm run build
```

2. **Проверьте папку `out/`:**
   После сборки там должны быть файлы:
   - `app/datalab.html`
   - `app/datalab/...`
   - `_next/...` (статические ресурсы)

3. **Скопируйте файлы на сервер:**
   Скопируйте ВСЁ содержимое папки `out/` в папку `/universe/` на сервере civilx.ru
   
   Структура должна быть:
   ```
   /universe/
     ├── app/
     │   ├── datalab.html
     │   └── datalab/
     ├── _next/
     └── .htaccess
   ```

4. **Убедитесь, что .htaccess скопирован:**
   Файл `.htaccess` должен быть в корне папки `/universe/` на сервере

5. **Проверьте права доступа:**
   Убедитесь, что веб-сервер имеет права на чтение файлов

## После развёртывания:

Приложение должно быть доступно по адресам:
- https://civilx.ru/universe/app/datalab
- https://civilx.ru/universe/app/datalab/project/proj-1

## Обновление:

При каждом изменении кода:
1. Запустите `npm run build`
2. Скопируйте новые файлы из `out/` на сервер
3. Очистите кэш браузера (Ctrl+Shift+R)
