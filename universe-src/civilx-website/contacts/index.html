Часто задаваемые вопросы

Как начать работу с CivilX?

Зарегистрируйтесь на сайте и начните с бесплатного пробного периода. Вы сможете создать до 3 проектов и изучить основные возможности платформы.

Какие форматы файлов поддерживаются?

CivilX поддерживает основные форматы документов: PDF, DOC, DOCX, а также изображения и CAD-файлы в популярных форматах.

Можно ли интегрировать с другими системами?

Да, CivilX предоставляет API для интеграции с внешними системами. Доступ к API включен в годовой тариф.
